# 基于AdminLET后台模板开发
## 数据库：Mysq5.7                                                                      
## 服务器：Tomcat9
## 开发工具：idea2019.2
## 前端：jsp，jquery
## 后端：Mybatis,Spring,Spring MVC,Spring Security 
## 实现功能：
1. 在spring security上使用BCryptPasswordEncoder对密码进行加密处理
2. 实现方法级权限控制
3. 方法级AOP日志
4. 多表操作，使用pagehelper分页查询
